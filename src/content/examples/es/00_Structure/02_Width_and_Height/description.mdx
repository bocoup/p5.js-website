  Patrón de líneas horizontales grises y verdes. La mitad izquierda también contiene líneas blancas verticales. La mitad izquierda está dividida en dos formas triangulares, una

Las variables 'ancho' y 'alto' contienen el ancho y alto de la ventana de visualización tal como se define en la función createCanvas().
